# Getting Started

## What EGGSISTENTIAL Is

EGGSISTENTIAL is a skill-based survival arcade game on Solana. Players connect a wallet, prepare a playable balance, enter live sessions, progress through checkpoints, and build Passport reputation through consistent play.

## Who This Is For

- players who want to understand the product flow
- partners who want to understand Passport utility
- developers who want to understand the technical stack

## Core Ideas

### Skill First

The experience is built around execution, timing, and progression.

### Wallet Native

The product starts from a Solana wallet, not a web2 account.

### Reputation As Output

Gameplay does not only create session results. It also creates Passport progression, which can become a reusable trust signal.

## Main User Flow

1. connect wallet
2. prepare playable balance in the vault
3. start a live session
4. progress through gameplay checkpoints
5. finish the session
6. build Passport reputation over time
