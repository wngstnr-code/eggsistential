# Gameplay

## Overview

EGGSISTENTIAL is built around a live session loop where player execution, timing, and consistency matter.

The goal is not only to complete a single session, but to build a believable history of progression over time.

## Core Session Flow

1. connect wallet
2. prepare playable balance in the vault
3. start a live session
4. move through the game and reach checkpoints
5. finish the session with a meaningful result
6. contribute to Passport progression

## What Makes The Experience Distinct

### Realtime Play

The product is designed around live gameplay rather than passive interaction.

### Checkpoint Progression

Sessions are structured around progression and decision points, giving each run a meaningful shape.

### Persistent Meaning

Each session is part of a longer player history. That makes gameplay progression relevant beyond a single run.

## Relationship To Passport

Gameplay is where Passport starts.

Passport is not an isolated feature. It is the long-term reputation output of repeated play, progression, and consistency.

This connection gives gameplay a second layer of meaning:

- immediate session performance
- long-term wallet reputation
