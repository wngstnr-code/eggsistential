# Eggsecutive Docs

EGGSISTENTIAL is a skill-based Solana arcade game where realtime gameplay, wallet-native balance flow, and Passport reputation are part of one connected product.

This documentation is written for both users and developers. It covers how the product works, how the system is structured, and where each part of the stack lives.

## Product Snapshot

- wallet-native gameplay on Solana
- vault-based playable balance flow
- backend-authoritative realtime sessions
- Passport reputation built from gameplay progression
- reusable trust layer for future integrations
